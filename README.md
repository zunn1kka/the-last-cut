# The Last Cut

Киноинформационный портал для поиска, оценки и обсуждения фильмов и сериалов.

## О проекте

"The Last Cut" - это веб-приложение, предназначенное для любителей кино, которые хотят находить информацию о фильмах и сериалах, оценивать просмотренный контент, делиться мнениями в комментариях и отслеживать свой прогресс просмотра. Проект разрабатывается в рамках дипломной работы и представляет собой полнофункциональный кино-портал с административной панелью и системой модерации.

## Функциональность

### Для всех пользователей

- Просмотр каталога фильмов и сериалов
- Поиск контента по названию
- Фильтрация по жанрам, году выпуска, рейтингу
- Детальная информация о фильмах и сериалах
- Просмотр информации об актёрах и съёмочной группе

### Для авторизованных пользователей

- Оценка контента по десятибалльной шкале
- Написание комментариев и ответы на них
- Лайки и дизлайки комментариев
- Добавление контента в избранное
- Отслеживание статуса просмотра (В планах, Смотрю, Просмотрено, Брошено)
- Личный профиль с историей оценок и комментариев
- Создание сборников фильмов

### Для модераторов

- Просмотр и удаление комментариев
- Обработка жалоб на комментарии

### Для администраторов

- Полное управление контентом (фильмы, сериалы, жанры)
- Управление персонами (актёры, режиссёры)
- Управление пользователями
- Модерация комментариев и жалоб

## Технологический стек

### Серверная часть

- NestJS - фреймворк для построения серверных приложений
- Prisma - ORM для работы с базой данных
- PostgreSQL - реляционная база данных
- JWT - аутентификация и авторизация
- Argon2 - хеширование паролей
- Nodemailer - отправка email-уведомлений
- Swagger - документация API
